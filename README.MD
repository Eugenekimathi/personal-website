# Eugene Kimathi — Personal Portfolio Website

A personal portfolio website built with HTML and CSS as part of the Moringa School software engineering bootcamp project.

---

## What It Does

The website introduces Eugene Kimathi as a junior developer and showcases his work. It is a single-page site with two main sections:

- **About Me** — a brief bio, skills overview, and stats
- **Projects** — a showcase of two completed projects

---

## Sections

### About Me
- Short bio describing who I am and where I am based
- Skills grouped into three categories: Languages, Tools, and Concepts
- Stats row showing projects built, months of training, and commitment level

### Projects
- **JS Calculator** — a JavaScript calculator that performs basic arithmetic and tracks calculation history
- **Cat Adoption Agency** — a responsive CSS layouts lab built with Flexbox and media queries

---

## Built With

- HTML5
- CSS3
- Google Fonts — Syne and DM Sans

---

## How To Run

1. Clone or download the repository
2. Make sure `index.html` and `style.css` are in the same folder
3. Open `index.html` with Live Server in VS Code
